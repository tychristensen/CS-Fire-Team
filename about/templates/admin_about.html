About

Intro

The Bridge ™ software is designed to integrate the data gathered by the Golden Fire Department and represent it in a meaningful way. As an administrator, you can update the status of personnel to reflect leave time, allowing them to generate an accurate LOSAP report. You can run reports on individuals, groups, or the entire staff to see where the department stands in terms of progress towards fulfilling requirements. Additionally, as an administrator you can upload data to the system at any time on the imports page and view overarching departmental statistics on the home page.

Home

This view will show you department-wide statistics including the proportion of staff that is on-track, behind, falling-behind, or complete with their requirements. You can customize the date-range for which the data is generated. Keep in mind, however, that the personnel requirement status data in the pie chart will always be generated as a year-to-date.

Reports

This feature is only available to officers and administrators. This view allows you to select a date-range, group of individuals, and report type and run a report. This report is then downloaded to your computer as a csv file. Standard reports contain information regarding calls (actual and total), shifts (including bonus shifts), work detail hours, sundays/dailies/weeklies, fundraisers, business meetings, and days/years of service. Additionally, there is a column indicating if the individual is on-track, behind, falling-behind, or complete with their requirements.

Custom Reports

To create a custom report, first select the date range of interest. Next, select the individual employee you would like to run the report on. Then, select the type of report you would like to generate. Then click the submit button. This will download a csv for the selected report type. The report will list all the data regarding the type selected from the database that we use to create the statistics. This function is mainly used to see a greater amount of details on the selected report type.

Personnel

This form is used to update a user’s status in the database. First select the employee you wish to modify from the drop down menu. Then from the next dropdown select that employee’s new status. You will be able to select a date in which you plan to make this status change go live. Lastly, there is a field to add a short note associated with the status change. Simply write this note in the given box. Once you have filled all the fields and are ready to change the employee’s status, simply press the ‘Submit’ button at the end of the page.

Import

This part of the website is used to upload csv files from When2Work and IamResponding into the database. IamResponding: First navigate to the IamResponding homepage. Click on subscriber login at the top right of the page. Once logged in, select the “Administrator Functions” button. You will be prompted to login again. Login, then once you are logged in go down to reports and select scheduling. Then fill in the necessary fields on the scheduling form and press the ‘Run Report’ button. Then in the newly opened report, click ‘Export to Excel’. This is the file you want to upload to the import page. Go to the import tab and select browse. Find the excel file you just downloaded and press the ‘upload’ button. The excel file will then be imported into the database.
